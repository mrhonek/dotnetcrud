Static Error Page

This is a static error page served from wwwroot/error.html.

If you can see this page, it means that static file serving is working, but the API may be experiencing issues.

Troubleshooting steps:

  1. Check the Railway logs for detailed error information
  2. Verify database connection strings are correct
  3. Check if Railway environment variables are properly set
  4. Try accessing the /health endpoint directly
